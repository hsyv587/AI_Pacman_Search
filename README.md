# AI_Pacman_Search
File documentation:
  all search agents are implemented in the file "pacmanAgent.py", which includes these search algorithms:
    RandomSequenceAgent
    RandomAgent
    GreedyAgent
    BFSAgent
    DFSAgent
    AStarAgent
    HillClimberAgent
    GeneticAgent
    MCTSAgent
  to run the game, just run "python pacman.py -p XXXAgent" with python 2.X
